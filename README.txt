SITE CARDÁPIO DIGITAL - FOOD PARK V8

Como usar:
1. Envie todos os arquivos desta pasta para uma hospedagem, Netlify, Vercel, Hostinger ou servidor próprio.
2. O QR Code deve apontar para o link do arquivo index.html, exemplo: https://seudominio.com.br/cardapio/
3. A página inicial mostra os dois cardápios: Braseiro/Drinks e Davi's Burger.
4. Os PDFs originais também estão na pasta /pdfs para download.

Arquivos principais:
- index.html: tela inicial com os dois cardápios.
- foodpark.html: cardápio Braseiro/Drinks.
- davis.html: cardápio Davi's Burger.
- style.css: visual do site.
